Overview

CSS allows you to adjust the position of elements with the position property.

There are five possible values for position:

  • Static
  • Relative
  • Absolute
  • Fixed
  • Sticky

These values, in combination with the top, left, right and bottom properties will determine the final location of the element.

Static

If position is not assigned a value, this the default position is static.

An element with static positioning is not affected by the top, left, right, bottom or z-index properties.

Instead, it follows the normal flow of the document.

Relative

Similarly with static positioning, relative positioning follows the normal flow of a document.

However, unlike static positioning, an element with relative positioning will be offset according to top, left, right, and bottom values.

This offset will not affect other elements. In other words, the space the element takes up within the page layout is the same as the static, non-offset position.

Relative position is also affected by z-index; that is, whether the element is forward or behind other elements.

Absolute

With absolute positioning, its designated space within the page layout is removed.

The element is instead anchored to its closest positioned ancestor or the initial containing block (the entire html page).

The final position is dictated by top, left, right, bottom and z-index values.

A common use case would be to anchor the absolute positioned element to a relative positioned element, like so:

              .parent {
                position: relative;
              }
    
              .child {
                position: absolute;
              }
Fixed

A fixed element is removed from the document flow and positioned according to the entire html page.

It will also stay in the same position on the viewport screen, regardless of where the user scrolls on the page.

The final position is dictated by top, left, right, bottom and z-index values.

Sticky

Elements with sticky positioning could best be described as an adaptable hybrid between relative and fixed positioning.

The element follows the normal flow of the document, except that it "sticks" to an ancestor which can scroll.

Sticky elements also change according to top, left, right, bottom and z-index values.
